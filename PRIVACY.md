# Privacy

AdPages Replacement Tool Opportunity Finder is local-only.

## Data Handling

- It reads local CSV or JSON files that you provide.
- It writes results to stdout or a local output file when requested.
- It does not make network calls.
- It does not scrape websites.
- It does not send outreach.
- It does not collect analytics.
- It does not require accounts, cookies, API keys, tokens, or credentials.

Input files may contain private backlink research, client domains, or prospect notes. Keep those files in your own controlled workspace and remove private client data before publishing examples.

## Outreach Safety

The tool only prioritizes possible replacement-tool campaigns. A human should manually verify every target, review the old linked tool, confirm the linking page context, and decide whether outreach is appropriate.
