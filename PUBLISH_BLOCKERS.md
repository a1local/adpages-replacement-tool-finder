# Promotion Blockers

The package can ship as a low-claim local CLI, but these items should be finished before a hosted public landing page, outreach campaign, or stronger benchmark claim.

- Validate CSV and JSON parsing against real exports from Ahrefs, Semrush, Screaming Frog, Sitebulb, browser crawlers, and Google Sheets.
- Add public sample exports that are either synthetic or explicitly cleared for publication.
- Add screenshot/report examples for the landing page.
- Validate scoring with at least 50 real candidate rows and tune thresholds based on human review.
- Add a short outreach ethics note to the public page: no scraping, no automated contact, no spam, and no exact-match anchor manipulation.
- Confirm any claims around backlink tools or resource-page workflows with current product export formats before publishing docs.
